Barra de Egresos

Introducción

La barra de egresos utilizada en los documentos como Vales de Salida, Envíos y Devoluciones se utiliza para ingresar detalles al documento y a la vez egresar producto del inventario una vez se guarde el documento. Ver figura 1.

Especificaciones

A continuación un cuadro con las especificaciones de cada campo de la barra:

Nombre Descripción Tipo Requerido
Cantidad Indica cuantas existencias va tener el detalle. Numérico entero mayor a cero. Ej.: 1, 10, 99. Si
Barra Es el código de barra del producto y se utiliza para identificar el producto ya sea utilizando el teclado o un lector de barras. Alfanumérico. Ej.: 123, 21AB323C. Si
Buscar Se utiliza para buscar al producto por su nombre en caso no se cuente con un código de barra. Los productos desactivados no aparecerán en la lista de resultados. Alfanumérico. Ej.: Aspirina, Leche. N/A

Modo de Empleo

Al igual que la barra de ingresos, su navegación son con las teclas Tab o Enter dependiendo del campo, un indicio de esto es que el fondo del campo se torna de color amarillo. La tecla Enter no funciona en todos los campos de la misma manera, en el campo "Cantidad" simplemente avanza hacia al siguiente campo. En el campo "Barra" no acepta la tecla Enter si no hasta que tenga un valor ingresado cambiando también el color de texto del botón "Agregar" (al final de la barra, ver figura 2) indicando que al presionar Enter el botón se presionara automáticamente, le dará ingreso al detalle y posicionara el cursor en el campo “Barra” listo para ingresar el siguiente detalle. No está de más mencionar que el campo cantidad siempre se mantiene con una cantidad igual 1 con el propósito de que si se utiliza el lector de barras se podrá ingresar producto por producto leyendo su código sin utilizar el teclado ni el mouse.


Figura 2.

Si en caso no se cuenta con el código de barra, se avanza al campo “Buscar” utilizando la tecla Tab y se ingresa el nombre del producto. El buscador desplegara una lista de resultados, se selecciona el resultado deseado ya sea utilizando el mouse o el teclado y el código de barra del producto seleccionado aparecerá en el campo “Barra” listo para presionar la tecla Enter.
