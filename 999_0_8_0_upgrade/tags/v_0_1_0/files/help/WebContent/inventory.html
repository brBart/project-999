Inventariados

Introducción

La sección de “Inventariados” en el modulo de inventario cuenta con herramientas para poder llevar a cabo inventarios físicos (inventariados) y luego compararlos con el inventario del sistema con el propósito de corregir diferencias. Para realizar un inventario se deben llevar a cabo 4 pasos de los cuales 2 son opcionales:

  1. Imprimir un selectivo para llevar un conteo físico de los productos de la tienda (opcional).
  2. Ingresar el conteo al sistema.
  3. Crear una comparación entre el conteo o inventario físico contra el inventario del sistema.
  4. Filtrar comparación para analizar mejor la información (opcional).

Para estos 3 pasos hay 3 herramientas en el sistema como lo muestra la figura 1.


Figura 1.

Selectivos

Se imprime un selectivo ingresando a la subsección “Selectivos” en la sección de “Inventariados” y se ingresan el rango de nombres de los productos o de casas (dependiendo del orden en que se desee el selectivo) de los cuales se desea llevar un conteo físico y se presiona el botón “Imprimir”. Ver figura 2.


Figura 2.

Luego la ventana de dialogo para imprimir aparecerá. Presionar el botón “Aceptar”.

Conteos

Imprimir un selectivo no es obligatorio ya que este es solo una ayuda para poder ir anotando los conteos físicos que se van realizando. No es el caso a la hora de ingresar los conteos realizados al sistema. Para esto se ingresa a la subsección “Conteos” en la sección de “Inventariados” y se presiona el link “Crear”. Ver figura 3.


Figura 3.

El formulario para crear conteos aparecerá. Se ingresa el motivo por la cual se llevo el conteo físico y se le da ingreso a los conteos utilizando la barra de conteo ya sea manualmente uno por uno o ingresando un archivo de texto en el caso de que se haya usado un recolector de barras (léase el manual Barra de Conteo). Al terminar presionar el botón “Guardar”. Ver figura 4.


Figura 4.

Luego de guardar el conteo el sistema generara automáticamente su número el cual no se podrá alterar. Ver figura 5.


Figura 5.

Comparaciones

El siguiente paso es crear una comparación entre lo que se conto físicamente y lo que el sistema tiene en su inventario. Para esto se ingresa en la subsección “Comparaciones” en la sección “Inventariados” y se presiona el link “Crear”. Ver figura 6.


Figura 6.

Luego el formulario para crear una comparación aparecerá. Se ingresa el número del conteo realizado anteriormente, el motivo y se selecciona si la comparación va ser contra todo el inventario en general o solamente contra los productos contados. Al final se presiona el botón “Guardar”. Ver figura 7.


Figura 7.

Luego la comparación realizada aparecerá en pantalla. La cual indicara los conteos físicos ingresados (Físico), las cantidades actualmente el sistema para cada producto (Sistema) y la diferencia. Ver figura 8.


Figura 8.

Filtrar Comparaciones

Por último se puede filtrar una comparación en caso esta sea demasiado grande o se quiere analizar la información en una forma más compacta. Para esta se ingresa a subsección “Comparaciones” en la sección de “Inventariados” y se presiona el link “Filtrar Comparación”. Ver figura 9.


Figura 9.

Luego el formulario para filtrar una comparación aparecerá. En el formulario se ingresa el número de la comparación a filtrar. Qué tipo de filtro aplicar y si se desea que despliegue los valores monetarios para cada detalle y el total. Para ver los valores monetarios se necesito ingresar con un rol con suficientes privilegios (ver privilegios en guía de referencia). Al final se presiona el botón “Aceptar”. Ver figura 10.


Figura 10.

Luego aparecerá la comparación con el filtro aplicado. Ver figura 11.
